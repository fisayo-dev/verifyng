# Team Name: Finsense

## Project Name- VerifyNG

VerifyNG is Nigeria's first AI-powered academic certificate verification platform. It solves a ₦50
billion annual problem: certificate fraud in hiring, admissions, and government employment.

### Deployment
[Live URL](https://verifyng-three.vercel.app)

## (Tech stack)
1. Web - Next.js + Typescript
2. Backend (API) - Python (Fast API)
3. Backend (ML) - Langgraph

## Setup
1. Frontend:
   - Clone the repository: `git clone https://github.com/fisayo-dev/verifyng.git`
   - Install dependencies: `npm install`
   - Get all your environment variables set up
   - Run the development server: `npm run dev`
2. Backend:
   - Change into the backend folder: `cd backend`
   - Install Python dependencies: `pip install -r requirements.txt`
   - Run the API: `uvicorn backend.main:app --reload`
